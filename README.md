# Backend server for Ride Sharing App

This backend server is created and designed using Node.js technology and MongoDB database(with Mongoose ORM).

## Key functionalities of the backend server:

1) **JWT-based Authentication** with protected API routes.
2) **Role based Authorization** for Driver, Passenger and Admin.
3) **API routes** for handling CRUD operations for the Ride Sharing Web App.
4) **Cloudinary** for Storage of Profile Images of Users and Retrieval.
5) **Socket.IO** for chat-based services like Real-time messages, read status, is typing indicator, etc.
6) Used open-street **maps API** for navigation based services like autocomplete, OSRM based distance calculation, geocoding and reverse geocoding.
7) Notification system for users.
8) **Admin based services (APIs)** for data analysis.

This project is a ride sharing based web app, with features like ride request, ride accept, ride search, create and read notifications, chatting app, admin services, etc.

To run the project, read the following steps:-
1) Pull the repo by downloading the ZIP file or by using the git pull "url" from your device.
2) Install the dependencies by running npm install inside the root of the project.
3) Run the project by running the command "npm run" in terminal at the root of the project directory.
4) The node.js backend server will be ready.
